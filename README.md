# Phishing Link Detection

This project detects phishing URLs using machine learning techniques.
It analyzes website links and predicts whether the URL is **legitimate or phishing**.

## Technologies Used

* Python
* Streamlit
* Scikit-learn

## How to Run

1. Install the required libraries:
   pip install -r requirements.txt

2. Run the application:
   streamlit run new26.py

## Project Files

* new26.py – Main application file
* phishing_model.pkl – Trained model
* phishing.csv – Dataset
* requirements.txt – Required libraries
